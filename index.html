Intro to Web Development

Twice weekly Code Nation class taught by working technology professionals volunteering their time & expertise

In Intro to Web Development, students learn front-end development basics like HTML and CSS, plus fundamental JavaScript syntax and concepts like variables, conditionals, and arrays.

Teacher-Led Programs (TLPs)

Code Nation curriculum taught by one of your teachers, with behind-the-scenes support from Code Nation instructional coaches, and once- or twice-weekly in-class support from working technology professionals volunteering their time and expertise

In a Code Nation TLP, students learn front-end development basics like HTML and CSS, plus fundamental JavaScript syntax and concepts like variables, conditionals, and arrays.

Fellowship

Once weekly project-based advanced after-school program taught by working technology professionals volunteering their time and expertise

Code Nation offers a Fellowship I program for students who have completed an Intro program, and a Fellowship II program for students who have completed a Fellowship I program.

In a Code Nation Fellowship I program, students learn advanced front-end development concepts like functions, objects, and APIs.

In a Code Nation Fellowship II program, students learn React.

Students graduate from every Code Nation program with a portfolio of projects they can use to apply for college, for internships, and for jobs.

After-School Clubs

Twice-weekly opt-in after-school clubs taught by working technology professionals volunteering their time and expertise

In a Code Nation after-school club, students are exposed to the same curriculum they would be in an in-school class: front-end development basics like HTML and CSS, plus fundamental JavaScript syntax and concepts like variables, conditionals, and arrays.

Affinity Groups

Affinity groups are spaces where students can meet other young people who share not only their passion for technology, but demographic similarities.

Code Nation offers affinity spaces and resources for...

  • Black students
  • Latinx students
  • Asian students
  • Girls & femmes
